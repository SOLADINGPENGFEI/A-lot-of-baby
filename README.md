## 积纳有品 本地部署

> 1. git clone https://github.com/SOLADINGPENGFEI/A-lot-of-baby.git
> 2. 安装依赖 cnpm install
> 3. 修改 package.json 文件中 proxy 字段的值 = 你本地服务器的 ip 地址 （"proxy": "http://127.0.0.1:8888"）
> 4. npm start
> 5. 部署到服务端 npm run build 中
